¿Qué es JavaScript?

JavaScript (abreviado comúnmente JS) es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos,3​ basado en prototipos, imperativo, débilmente tipado y dinámico.

Se utiliza principalmente en su forma del lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas4​ aunque existe una forma de JavaScript del lado del servidor (Server-side JavaScript o SSJS).

¿Qué nos permite hacer JS?

JavaScript nos permite realizar multiples acciones haciendo uso de objetos, diferentes variables, eventos, entre otros. Para de esta manera generar páginas más dinámicas para el usuario

Arreglos

Los arreglos son un tipo de estructura de datos que nos permite almacenar múltiples variables en un solo conjunto ordenado por posición. La sintaxis es la siguiente:

var cars = ["Saab", "Volvo", "BMW"];

Arreglo Impreso

Al igual que el resto de variables los arreglos cuentan con funciones determinadas, con estas funciones se pueden realziar acciones como agregar,eliminar y modificar posiciones.

Función pop

Esta funcion permite borrar el último elemento del arreglo. Ejemplo:

cars.pop();

Función push

Esta funcion permite agregar un elemento al final del arreglo. Ejemplo:

cars.push("Audi");

Ordenar Arreglo

Un arreglo se puede ordenar a través de la función sort (ascendente) o reverse (descendente). Ejemplo:

Banana,Orange,Apple,Mango
