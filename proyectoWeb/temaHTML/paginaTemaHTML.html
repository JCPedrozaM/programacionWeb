Tema HTML

HTML (Hypertext Markup Lenguage) o "Lenguaje de marcado de hipertexto" es un lenguaje de etiqueta predefinidas que le indican al navegador o browser el tipo de contenido que debe mostrar

Versiones:
  1. HTML 1.0
  2. HTML 2.0
  3. HTML 3.0
  4. HTML 4.0
  5. HTML 5.0


HTML 1.0

Primer versión de HTML, consistía de etiquetas básicas, el formato o estilo se mezcclaba en el mismo código HTML.
